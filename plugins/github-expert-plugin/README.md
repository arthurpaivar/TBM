# GitHub Expert Plugin

Specialist in GitHub repository analysis, management, and best practices.

## Skills
- **GitHub Repository Specialist**: Deep understanding of repository structure, code, PRs, issues, workflows, and health metrics.

## Commands
- `/create-github-element` — Create new elements in GitHub (repos, issues, PRs, branches, workflows)
- `/update-github-element` — Update existing GitHub elements
- `/remove-github-element` — Remove GitHub elements
