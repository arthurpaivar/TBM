Conduct a comprehensive technology assessment on $ARGUMENTS using the technology-expert-plugin skills.

Follow this process:
1. Load the relevant specialist skill(s) from this plugin based on the topic
2. Apply the SPAR Framework: Sense the context → Plan the approach → Act on analysis → React with recommendations
3. Use the Pyramid Principle: lead with conclusions, then supporting evidence
4. Structure findings using MECE decomposition
5. Deliver an executive-ready assessment with clear recommendations

If the topic involves architecture, use the architecture-specialist skill.
If it involves agile/delivery metrics, use the agile-specialist skill.
If it involves AI/agents, use the ai-specialist skill.
If it involves DevOps/SDLC, use the tech-delivery-specialist skill.
If it involves strategy/transformation, use the tech-strategy-advisory skill.

Output a structured assessment document with: Executive Summary, Current State, Gap Analysis, Recommendations, and Next Steps.
