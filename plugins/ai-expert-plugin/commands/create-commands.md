Create new commands for $ARGUMENTS using the command-creation-specialist skill.

Follow this process:
1. Load the command-creation-specialist skill from this plugin
2. Define the command purpose, name, and parent skill
3. Select appropriate tools for the command
4. Design the process steps (5-7 steps)
5. Write the command .md file with frontmatter and instruction body
6. Validate against the standard command template

Output: Complete command .md file(s) ready for installation in the target plugin.
