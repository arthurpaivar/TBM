Create a new skill for $ARGUMENTS using the skill-creation-specialist skill.

Follow this process:
1. Load the skill-creation-specialist skill from this plugin
2. Define the skill domain, trigger keywords, and scope
3. Check for overlap with existing skills
4. Design the SKILL.md structure following the standard template
5. Write the complete SKILL.md with frontmatter, core principles, capability map, output patterns, and quality checks
6. Validate trigger keyword coverage and actionability

Output: Complete SKILL.md file ready for installation in a plugin.
