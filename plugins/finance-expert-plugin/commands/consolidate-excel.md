Consolidate and analyze the Excel data in $ARGUMENTS using the tech-bm-specialist skill.

Follow this process:
1. Load the tech-bm-specialist skill from this plugin
2. Read and assess the Excel file or cost data provided
3. Apply TBM Framework 2.0 and TBM Taxonomy v5.0.1 for categorization
4. Consolidate data into standard cost pools (Run vs. Grow vs. Transform)
5. Calculate unit costs and total cost of ownership where applicable
6. Identify anomalies, trends, and optimization opportunities
7. Generate a consolidated summary with visualizations

Output: Consolidated Excel Summary, TBM Categorization, Key Metrics, Anomalies/Trends, and Optimization Recommendations.
