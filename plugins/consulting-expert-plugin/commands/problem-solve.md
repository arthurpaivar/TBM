---
description: Structured problem decomposition and analysis
allowed-tools: Read, Write, Edit, Grep, Glob, Bash, WebSearch, WebFetch, Task
argument-hint: [problem to solve]
---

Apply structured problem-solving frameworks to $ARGUMENTS using the problem-solving-specialist skill.

Follow this process:
1. Load the problem-solving-specialist skill from this plugin
2. Frame the problem using SCQA (Situation, Complication, Question, Answer)
3. Decompose using MECE issue trees
4. Generate and test hypotheses
5. Apply 5 Whys or Ishikawa for root cause analysis where appropriate
6. Evaluate options using decision matrices

Output: Problem Statement, Issue Tree, Hypotheses, Root Cause Analysis, Options with Trade-offs, Recommendation.
