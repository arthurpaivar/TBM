# Consulting Expert Plugin

Your AI-powered consulting partner for structured problem solving, deep research, and industry benchmarking. Three daily-use commands orchestrate specialist skills across the entire plugin ecosystem — combining consulting methodology with technology and finance domain expertise for executive-grade outputs.

Built on the **SPAR agent framework** (Sense → Plan → Act → React) and **Pyramid Principle** logic. Commands use subagent workflows to invoke specialists from technology-expert-plugin and finance-expert-plugin for domain depth.

## Commands (Daily Use)

| Command | What It Does | Key Feature |
|---------|-------------|-------------|
| `/problem-solve` | McKinsey/BCG-grade problem decomposition → issue tree → root cause → recommendation | Dispatches domain specialist subagents to analyze priority branches |
| `/find-references` | Deep multi-source, multi-modal research with curated references | Parallel search streams for articles, videos, tools, communities — better than Google |
| `/find-benchmark` | Industry benchmarks from Gartner, Forrester, McKinsey with gap analysis | Multi-source benchmark research with normalization and improvement roadmap |

### Communication Protocol (All Commands)

Every command output follows the same structure:
- **Opening (The So-What)**: Lead with the answer — recommendation, synthesis, or verdict
- **Body (The Evidence)**: Structured analysis supporting the opening
- **Conclusion (Insight & Action)**: Strategic insight, specific next steps, monitoring recommendations

## Cross-Plugin Architecture

This plugin is designed to orchestrate skills from the broader ecosystem:

```
consulting-expert-plugin (orchestration layer)
├── technology-expert-plugin (domain depth)
│   ├── architecture-specialist → system design, patterns, trade-offs
│   ├── tech-strategy-advisory → strategic frameworks, transformation
│   ├── ai-specialist → AI/ML capabilities, implementation
│   ├── agile-specialist → delivery methodology, agile metrics
│   └── tech-delivery-specialist → CI/CD, DevOps, DORA
├── finance-expert-plugin (financial depth)
│   ├── tech-bm-specialist → TBM taxonomy, cost allocation
│   ├── it-portfolio-management → portfolio rationalization, TIME model
│   └── excel-for-finance → financial modeling
└── consulting-expert-plugin (own skills)
    ├── problem-solving-specialist → analytical engine
    ├── deep-search-specialist → research engine
    ├── industry-benchmark-specialist → benchmarking engine
    └── business-case-specialist → financial justification engine
```

## Core Principles (All Skills)

- **SPAR Framework**: Sense → Plan → Act → React
- **Pyramid Principle**: Lead with the answer, structure top-down
- **SCQA Pattern**: Situation → Complication → Question → Answer
- **MECE Decomposition**: Mutually Exclusive, Collectively Exhaustive
- **Confidence Calibration**: Explicit confidence levels
- **Source Attribution**: Always cite sources with credibility ratings
- **Practical & Adaptable**: Executive-ready, immediately actionable outputs

## How to Use

Invoke commands directly with your topic:

- `/problem-solve Why are our deployment lead times increasing?`
- `/find-references event-driven architecture for payments`
- `/find-benchmark DORA metrics for enterprise financial services`

Each command accepts any input format — a phrase, a question, a paragraph, or an image. The command will parse the intent and execute the full workflow.

## Setup

No configuration needed. Install the plugin and all skills and commands are available immediately. For full cross-plugin orchestration, ensure technology-expert-plugin and finance-expert-plugin are also installed.
