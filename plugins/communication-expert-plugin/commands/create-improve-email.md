1. Load the e-mail-generator skill from this plugin
2. If improving an existing draft, assess clarity, tone, and structure
3. Clarify: audience, purpose, tone, and key messages
4. Apply the Pyramid Principle: lead with the key message/ask (BLUF)
5. Structure with clear subject line, greeting, body, action items, and closing
6. Calibrate tone for the target audience (executive, peer, team, external)
7. Apply cultural sensitivity for LATAM contexts where applicable
