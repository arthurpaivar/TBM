Create an executive summary for $ARGUMENTS using the text-improver-specialist and e-mail-generator skills.

Follow this process:
1. Load the text-improver-specialist and e-mail-generator skills from this plugin
2. Identify the key messages, decisions, and actions from the source material
3. Apply the SCQA Pattern (Situation, Complication, Question, Answer)
4. Structure using the Pyramid Principle: lead with the conclusion
5. Keep to 1 page maximum — concise, data-driven, action-oriented
6. Calibrate tone for executive audience
7. Include clear next steps and decision points

Output: Executive Summary with Situation, Key Findings, Recommendation, and Next Steps.
