Create or improve a presentation on $ARGUMENTS using the powerpoint-specialist skill.

Follow this process:
1. Load the powerpoint-specialist skill from this plugin
2. If improving, assess current deck structure, clarity, and visual hierarchy
3. Clarify: audience, purpose, time constraint, and key messages
4. Apply the Pyramid Principle for narrative flow
5. Use SCQA (Situation, Complication, Question, Answer) for executive presentations
6. Design slide structure with clear titles, key messages, and supporting data
7. Include speaker notes and talking points

Output: Presentation Outline, Slide-by-Slide Structure (title, key message, content, speaker notes), and Design Recommendations.
