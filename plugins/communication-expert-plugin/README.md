# Communication Expert Plugin (v1.0.0)

Your AI-powered coworker for professional communication in technology leadership at JP Morgan LATAM. Four specialist skills and four commands covering email drafting, EN↔PT-BR translation, presentation structuring, and text quality improvement.

Built on the **SPAR agent framework** (Sense → Plan → Act → React) and **Pyramid Principles** logic (Barbara Minto).

## Skills

| Skill ID | Skill | Focus Areas |
|----------|-------|-------------|
| **2.1** | **E-mail Generator** | Status updates, escalations, approvals, follow-ups, stakeholder comms, BLUF methodology |
| **2.2** | **Translation Specialist** | EN↔PT-BR professional translation, tech terminology, cultural adaptation |
| **2.3** | **PowerPoint Specialist** | Presentation structure, slide design, storytelling for leadership, deck templates |
| **2.4** | **Text Improver Specialist** | Clarity, conciseness, tone, structure, audience-aware rewriting |

## Commands

| Command | What It Does |
|---------|-------------|
| `/draft-email` | Draft a professional corporate email with pyramid principle and cultural sensitivity |
| `/translate` | Professional EN↔PT-BR translation with terminology accuracy |
| `/improve-text` | Refine and elevate professional writing for clarity, tone, and structure |
| `/presentation` | Structure a presentation using pyramid principle and SCQA framework |

## Core Principles (All Skills)

Every skill follows these governing principles:

- **SPAR Framework**: Sense → Plan → Act → React
- **Pyramid Principle**: Lead with the answer, structure top-down
- **SCQA Pattern**: Situation → Complication → Question → Answer
- **MECE Decomposition**: Mutually Exclusive, Collectively Exhaustive
- **Confidence Calibration**: Explicit confidence levels, escalate when uncertain
- **Source Attribution**: Always cite frameworks and data sources
- **Practical & Adaptable**: Immediately usable, context-aware outputs

## How to Use

Each skill activates when you mention relevant topics, or use the slash commands for structured workflows:

- `/draft-email escalation to CIO about vendor delays` → Professional email with BLUF methodology
- `/translate executive summary to Portuguese` → Professional EN→PT-BR translation
- `/improve-text quarterly report draft` → Refined writing with clarity and tone improvements
- `/presentation QBR technology update` → Structured deck using Pyramid Principle and SCQA

## Setup

No configuration needed. Install the plugin and all 4 skills and 4 commands will be available automatically.
