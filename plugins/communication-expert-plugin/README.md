# Communication Expert Plugin

Your AI-powered coworker for professional communication in technology leadership at JP Morgan LATAM. Four specialist skills covering email drafting, EN↔PT-BR translation, presentation structuring, and text quality improvement.

Built on the **SPAR agent framework** (Sense → Plan → Act → React) and **Pyramid Principles** logic (Barbara Minto).

## Skills

| Skill ID | Skill | Focus Areas |
|----------|-------|-------------|
| **2.1** | **E-mail Generator** | Status updates, escalations, approvals, follow-ups, stakeholder comms, BLUF methodology |
| **2.2** | **Translation Specialist** | EN↔PT-BR professional translation, tech terminology, cultural adaptation |
| **2.3** | **PowerPoint Specialist** | Presentation structure, slide design, storytelling for leadership, deck templates |
| **2.4** | **Text Improver Specialist** | Clarity, conciseness, tone, structure, audience-aware rewriting |

## Core Principles (All Skills)

Every skill follows these governing principles:

- **SPAR Framework**: Sense → Plan → Act → React
- **Pyramid Principle**: Lead with the answer, structure top-down
- **SCQA Pattern**: Situation → Complication → Question → Answer
- **MECE Decomposition**: Mutually Exclusive, Collectively Exhaustive
- **Confidence Calibration**: Explicit confidence levels, escalate when uncertain
- **Source Attribution**: Always cite frameworks and data sources
- **Practical & Adaptable**: Immediately usable, context-aware outputs

## How to Use

Each skill activates when you mention relevant topics. For example:

- "Draft an escalation email to the CIO" → E-mail Generator
- "Translate this executive summary to Portuguese" → Translation Specialist
- "Help me structure a presentation for the QBR" → PowerPoint Specialist
- "Improve the tone and clarity of this report" → Text Improver Specialist

## Setup

No configuration needed. Install the plugin and all 4 skills will be available automatically.
