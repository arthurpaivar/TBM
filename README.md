# TBM - Technical Business Manager Repository

This repository contains versioned plugins and skills for Claude AI.

## Structure

- `skills/public/` - Public skills (docx, pdf, pptx, xlsx, etc.)
- `skills/private/` - Private/custom skills
- `skills/examples/` - Example skills for reference

## Workflow

1. Draft updates in Obsidian
2. Finalize and approve changes
3. Deploy via Claude Cowork automation
4. Git maintains full version history for rollbacks

## Deployment

All deployments are automated through Claude Cowork with semantic commit messages.
